Project Title¶

Webscraping Real Estate Websites for Housing Market Analysis

Project Summary

In this project we will scrape the REMAX website (https://www.remax.ca/) to get information of the housing properties up for sale in different cities across different provinces in Canada. We will be using Selenium to assist in scraping the REMAX for both the basic and detailed information of each listing across different pages. We will use this scraped data do the following:

  • Map these listings to quickly identify regions based on their price

  • Plot distributions of housing prices across different regions for comparison

  • Visualize housing price distribution based on housing type (condos, single family etc.)

  • Find average price of listing across different cities and identify most and least expensive cities

  • Identify top 10 most expensive and top 10 least expensive listings

  • Analyze Price vs Square Footage trend for each region and identify any similarities or discrepancies

List of steps followed:

  • Step 1: Gather basic and details data from REMAX website for various cities and save them locally
  • Step 2: Append all the saved scraped data as one pandas dataframe
  • Step 3: Merge basic and details data on unique #MLS key, perform EDA and treat missing data
  • Step 4: Correct data types , extract new features and remove redundant or misleading data
  • Step 5: Use geopy library to extract latitude and longitude coordinates using the address of each listings and add it to dataframe
  • Step 6: Create a dashboard using Plotly and Dash to answer all the questions
  • Step 7: Deploy the dashboard on heroku (https://housingmarketanalysis.herokuapp.com/)

Import *custom_functions* and *app_layouts* Function¶

**custom_functions** and **app_layouts** functions are save in functions folder in the project directory.

  • **custom_functions** contains following functions:

    • web_scrape: This function scrapes the listing information of real estate properties from REMAX website and store the into two seperate lists basic_info and detail_info lists
    • listtodataframe: Function to convert scraped list of basic_info and detail_info in to respective dataframes
    • missing_data_stats: Reads a dataframe and generates its statistics and missing value tables
    • sqft_area: Takes square footage column as an input and extracts numbers from it where a range is reported like 600-700 sqft an average of these numbers is taken
    • area_from_dims: Function to generate area from room dimension tables
    • latlong: Function to convert address to lat long coordinates
  • **app_layout** contains following function:
    • pe_8_grid_3cols_layout: Function to generates a 8 grid layout of dash web app spread across three columns. The 1st column has two rows, the 2nd and 3rd each have 3 rows.

Correct Data Types , Extract Features and Clean Data¶

#------------------------------------------------------------------------------
# Trim all columns and Replace NaNs for room dimension columns
#------------------------------------------------------------------------------
remax = remax.apply(lambda x: x.str.strip() if x.dtype == "object" else x)

# Replac NaNs with empty strings
remax['Bedroom'].fillna('', inplace =True)
remax['Dining Room'].fillna('', inplace =True)
remax['Living Room'].fillna('', inplace =True)
remax['Kitchen'].fillna('', inplace =True)

#------------------------------------------------------------------------------
# Add Extra Feature Columns
#------------------------------------------------------------------------------
# Add Province Name Column
remax['Province'] = remax['Address'].apply(lambda x:x.split(',')[-1].strip())

# Add City Name Column
remax['City'] = remax['Address'].apply(lambda x:x.split(',')[-2].strip())

# Add Street Name Column From Address
remax['Street'] = remax['Address'].apply(lambda x:x.split(',')[-3])
remax['Street'] = (remax['Street'].apply(lambda x:
                        (re.findall('(-?\d*.*\w*)',x)[0]).split('-')[-1].strip()))

# Create a Street_City column to later use for lat, long coordinates
remax['Street_City'] = remax['Street'] + ' , ' +remax['City']

# Extract Property Tax Amount from Property Tax attribute
remax['Property Tax'] = remax['Property Tax'].replace('N/A',np.NaN)
remax['Property Tax Amount'] =  (remax['Property Tax'].apply(lambda x:str(x).split('(')[0].
                                                        replace('$','').replace(',','').
                                                            strip()))

remax['Property Tax Amount'] = remax['Property Tax Amount'].astype(float)

# Add area features using custom functions
remax['Total_Sqft'] = cf.sqft_area(remax, 'Square Footage')
remax['Bedroom_Sqm'] = cf.area_from_dims(remax,'Bedroom')
remax['Dining_Sqm'] = cf.area_from_dims(remax,'Dining Room')
remax['Living_Sqm'] = cf.area_from_dims(remax,'Living Room')
remax['Kitchen_Sqm'] = cf.area_from_dims(remax,'Kitchen')

#------------------------------------------------------------------------------
# Convert data types and Drop redundant columns and Rows
#------------------------------------------------------------------------------
# Convert Price to Integer
remax['Price'] = remax['Price'].apply(lambda x:int(x.replace('$','').replace(',','')))

# Convert date objects to datetime
remax['Date Listed'] = pd.to_datetime(remax['Date Listed'])
remax['Last Updated'] = pd.to_datetime(remax['Last Updated'])

#------------------------------------------------------------------------------
# Column Operations
#------------------------------------------------------------------------------
# Keep only these relevant columns

                'Street_City', 'Property Type','Date Listed', 'Price',
                'Beds', 'Baths', 'Total_Sqft', 'Bedroom_Sqm', 'Dining_Sqm',
                'Living_Sqm', 'Kitchen_Sqm',
                
                'Cooling', 'Heating','Parking', 'Basement',
                'Amenities', 'Features', 'Exterior Features'])

remax = remax[filter_list]

# Row Operations

remax.drop(remax.loc[remax['Price']<100000].index, inplace=True)

    dff = dff[dff[checkbox_filter_property].isin(list(checkbox_id))]
    
    #---------------------------------------------------------------------------------
    # Create all the required plots to populate on callback
    #---------------------------------------------------------------------------------
    px.set_mapbox_access_token('pk.eyJ1Ijoid2NkLXBhcnJ5IiwiYSI6ImNrZ2F3emExaTA1MTMyem1xYWlxaTJmOXQifQ.uNDZBfkfSYD1JxGsW94fzg')
    fig1 = px.scatter_mapbox(dff, 'Latitude', 'Longitude',color='Region',size='Price',
                             template='custom',hover_name = 'Address',zoom=3)
    
    fig2 = px.violin(dff,'Region','Price',color='Region',template='custom', points = False,

    fig3 = px.box(dff, 'Price', 'Property Type', color='Property Type',template='custom',
